Filter, Sort, Map, and Reduce all loop over the data. All four higher-order functions use a callback. http://javascriptissexy.com/understand-javascript-callback-functions-and-use-them/ Callback: Derived from a programming paradigm known as functional programming. At a fundamental level, functional programming specifies the use of functions as arguments. Callback 2: a.k.a. a higher-order function, is a function that is passed to another function as a parameter, and the callback function is called (or executed) inside the main function.
